Ambrosia - Responsive Bootstrap Theme

This is a Twitter Bootstrap Theme perfectly suitable for Web Applications or Corporate Websites.

Features:

* 6 Page Templates
* Responsive Design
* Retouched Buttons Set
* Full-width header and footer
* Font Awesome Icons
* Google Webfonts
* 3 Color skins (green, blue & red)
* 8 Background images
* Featuring lots of love and Bootstrap v2.1.0

Color Skins:

I have included the 3 css skins to somewhat match your project identity and serve you as a good starting point:
* green.css (defaul)
* blue.css
* red.css
Choose one of those and use it as your main skin. Unlink unused stylesheets at head section of all html files.

Important Now that you've got so far it's time to get rid of the "Style Switcher" stuff:
* Delete file: css/styleswitcher.css
* Delete file: js/styleswitcher.js
* Unlink style switcher CSS or JS files at 'head' section of all html files
* And last but not least, delete style switcher hyperlinks of all html files (delete the whole div).
* Thats it, now you are done :)

Backgrounds:

I have included 8 backgound images for the header section (look up in 'assets' folder).
You can switch between them editing header background in base.css file.

Responsive Videos:

In case you might want to use videos with this template I have included necesary css to make them responsive.
Videos must be wrapped in a .video-container div which will adopt the width of its immediate parent. So you may want to use a basic set of .row and .span to define parent's width. For example:

  <div class="row">
   <div class="span4"><!--given width-->
    <div class="video-container"><!--adaptive container-->
      <iframe src="url"></iframe><!--video-->
    </div>
   </div>
  </div>
